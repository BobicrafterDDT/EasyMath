User manual

This is a representation of the engine EasyMath.

What is this?

This is a java compiled file, that is a multi-functional program
that makes your life easier.

How to use modes?

There are many modes to choose from.And in here is explained how
to use each mode, but how to get to them? Well... When you run
the Engine, a window dialog will pop up. In that window you`ll
find three buttons: a start math? button, a Other
programs button and a close button.
  • How to get to the modes?
    When you run the Engine, a window dialog will pop up.
    In that window you`ll find three buttons: a start math? button,
    a start image writer? button and a close button.
    The start math? button opens a menu dialog for all of
    the math modes. The start image writer button opens you
    a window dialog for the ImageWriter program.

  • Find biggest General Divisor with two numbers
    (FBGD with two numbers)
    To get in this mode, you have to go start math? >
    FBGD with two numbers. When you click the button
    a window will pop up with two text fields, a Ready?
    button and a go back button. Now all you have to do
    is to insert the first number and the second number that is
    needed in the calculation. Then you press the Ready?
    button, it will hide and another button will pop up: Run
    button. You click that too, and a text will appear next
    to the button. That is your result. Now if you want to go back,
    you just press the go back button.

  • Find biggest General Divisor with three numbers
    (FBGD with three numbers)
    To get in this mode, you have to go start math? >
    FBGD with three numbers. It`s nearly as the same
    as FBGD with two numbers, but now there are three text
    fields. The rest of it is the same. The difference here
    is that you have to insert three numbers and that`s it!
    You press the Ready? button and then the Run
    button. The result is the text besides the Run button.
    Then if you want to go back you press the go back button.

  • FFSR (Find Face of a Square or a Rectangle)
    To get in this mode, you have to go start math? >
    FFSR. In here you have to type in the two sides of your
    square/rectangle and then you`ll get the face of that.

  • FTSR (Find Tour of a Square or a Rectangle)
    To get in this mode, you have to go start math? >
    FTSR. In this mode you type in the two sides and you get
    the tour of your square/rectangle.

  • FEON (Find Amount Of Numbers)
    To get in this mode, you have to go start math? >
    FEON. In this mode you have to insert the start range
    and end range of your numbers. Example:
    1; 100
    One is the start range and a hundred is the end range. It will
    give you an output of:
    100
    This is how many numbers there are from on to 100 including
    1 and 100.

The next one modes have the same insert mode, but different calculations.
  • Find Face of a Square or a Rectangle(FFSR)
    To get in this mode, you have to go start math? >
    FFSR. You insert the two sides of the square/rectangle,
    then you press the buttons Ready? then Run and
    your result is on the side of the Run button. If you
    want to go back, you press the go back button.

  • Find Tour of a Square or a Rectangle(FTSR)
    To get in this mode, you have to go start math? >
    FTSR. You insert the two sides of the square/rectangle,
    then you press the buttons Ready? then Run and
    your result is on the side of the Run button. If you
    want to go back, you press the go back button.

  • Gaus
    To get in this mode, you have to go start math? >
    Gaus. You insert the start number and the end number
    of your exercise. Make sure that the start number
    is not bigger than the end number!!!

  • Find Lowest Multiple (LCM)
    To get in this mode, you have to go start math? >
    LCM. Here you enter the two numbers of which you
    want to find LCM.

  • Find random number (random)
    To get in this mode, you have to go start math? >
    Random. Here you only type in the start and end
    range.

Now, let`s look at the Other programs button. When you click it, a
menu will pop up, with all of the other programs there are that are doing
other things rather than only math. It depends on what version of the Engine
you are, because specifically in this menu, are the main changes. In the first
version you have only two buttons (programs): start image writer? button
(program) and a start To-Do list maker? button (program).
  • Image Writer
    To get to this mode you have to go to Other programs >
    start image writer?. When you enter the menu, you will
    find two text fields. In the first one you have to insert the
    text file path. In the second one you have to insert the output
    file location.Note that:
    > You have to insert both file paths for the very least start
    on which disk you are. Example:
    txt file path: C:/documents/MyFile.txt
    output file path: C:/documents/Out.jpg
  • To-Do list maker
    To get to this mode you have to go to Other programs >
    Start To-Do list maker?. When you enter the menu, you will
    see quite a bunch of things. There will be 3 buttons and a text
    field(fields for when Replace button is pressed). Here are
    the explanations for all of them:

    • Add (item)
      With this button you can add an item to your list. To add
      an item, you have to type in what do you want to add and
      then you click the Add (item) button. Then
      your item will be displayed as a text in the lower part
      (below the field).

    • Remove (item)
      With this button you can remove an item from your list. To
      remove an item you have to type the item you want to remove
      and then you click the Remove (item) button. Then
      the item you removed will disappear from the text in the lower
      part (below the field).Be sure that the item you want to
      remove is exactly the same as the item in the list or else the program won`t
      work properly!!!
